# Called from a portal button in layout FAT Link
If [ FAT Link::Number of instances >0 ]
	Show Custom Dialog [ Title: "Cannot delete Link" ; Message: "This Link is used by one or several Page items. Remove all references to this Link first. " ; Default Button: "OK" ]
Else If [ FAT Link::Number of instances =0 ]
	Delete Portal Row [ With dialog: OFF ]
End If
