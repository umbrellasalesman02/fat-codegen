# Called from IO Type
Freeze Window
Set Error Capture [ ON ]
Perform Script [ From list ; "Save Active Layout" ]
Set Variable [ $io_type[1] ; IO Type::FATType ]
Go to Layout [ Layout: "FAT Block" ; Animation: None ]
Perform Find [ ⚠️ RESTORE ⚠️ ]
Extend Found Set [ ⚠️ RESTORE ⚠️ ]
Sort Records [ With dialog: OFF ; ⚠️ PARAMETER "Restore" NOT PARSED ⚠️ ; ⚠️ PARAMETER "SortSpecification" NOT PARSED ⚠️ ]
If [ Get(foundCount)=0 ]
	Show Custom Dialog [ Title: "No Blocks found" ; Message: "It seems there's no Blocks in any Title using this IO type. " ; Default Button: "OK" ]
	Perform Script [ From list ; "Go previous layout" ]
End If
